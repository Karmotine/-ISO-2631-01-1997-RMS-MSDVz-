# -ISO-2631-01-1997-RMS-MSDVz-
运用ISO 2631-01-1997标准中的 时域 方法计算 加权平均加速度(RMS)与晕车剂量值(MSDV)，包含计算x,y,z三轴分别的加权平均加速度、三轴合成加权平均加速度、晕车剂量值MSDVz。
该计算是我本人第一次尝试编写的，使用了AI辅助我写作。本人不是相关物理、交通、生物医学等专业的学生，对标准及其中的指标理解会有不透彻的地方，敬请指出
使用方法：直接下载py文件，运行前确保已拥有一系列数据，包括x，y，z轴加速度（时域），明确你的数据的回传速率（频率）是多少。
对200hz的数据，采用了降频器将其降到20hz进行计算，避免出现NaN错误
